# ml-for-cybercrime
Source code for building and classifying Brazilian cybercrime judicial data using topic modeling and supervised machine learning
This repository contains the source code and supplementary resources developed for a research project on the application of machine learning to Brazilian cybercrime judicial data. The project implements a complete workflow including data collection, text preprocessing, topic modeling, and supervised text classification.

## Repository Contents

### `01_extracao_dados.R`

Collects and extracts the raw judicial data used throughout the study.

### `02_preprocessamento.R`

Performs data cleaning and textual preprocessing, including:

This script uses the auxiliary files located in the `data/` directory.

### `03_modelagem_topicos.R`

Performs topic modeling using **Latent Dirichlet Allocation (LDA)

### `04_classificacao_tfidf.ipynb`

Implements the supervised classification experiments using **TF-IDF** document representations. The notebook includes feature extraction, model training, performance evaluation, and result visualization.

### `04_classificacao_bertimbau.ipynb`

Implements the supervised classification experiments using contextual embeddings generated by **BERTimbau**. The notebook includes text encoding, model training, evaluation, and visualization of the experimental results.

### `requirements.txt`

Lists the Python dependencies required to execute the Jupyter notebooks.
